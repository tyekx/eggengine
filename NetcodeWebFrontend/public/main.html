What is Netcode?

Netcode is a first person shooter game developed by us, Czövek Márton and Musa Benedek throughout our journey in university.

This project started out as a collective test for every piece of game development and computer graphics we learned about. The important parts of the project was coded by us with the usage of some libraries.

The website you are currently reading was created as a multiplayer extension for the project. Basically a deep dive for how to create a multiplayer game from the registration to the actual communication between clients and server while gaming. While the majority of the work was done by me, I had a lot of help, please continue reading to get to know my collaborators

- Musa Benedek

SchDesign

Some of the models were made by Schőnherz Design, at first we tried to create an art pipeline, but it quickly became apparent, that the work required would outweight the value of the solution. Therefore we settled for the oldschool model export-import method. It was rather interesting to work with blender the first time after so many years of 3DS Max. Be sure to give look them up if you are looking for anything design related question at schdesign.hu. Shoutout to Zoltán Ábrahám for the infinite patience and help.
