What is Netcode?

Netcode is a first person shooter game developed by us, Czövek Márton and Musa Benedek throughout our journey in university.

This project started out as a collective test for every piece of game development and computer graphics we learned about. The important parts of the project was coded by us with the usage of some libraries.

The website you are currently reading was created as a multiplayer extension for the project. Basically a very in-depth test for how to create a multiplayer game from the registration to the actual communication between clients and server while gaming. While most of the project was a collaboration this project was made by me.

- Musa Benedek
